Featured Role

Junior Web Developer

📍 Enfield, London (Hybrid) ⏱ Full-time 💰 £26,000 – £30,000 p/a 🎓 Entry Level ✦ Actively Hiring

We're hiring junior web developers to join our growing team at Enfield Dev Ltd. This is an excellent opportunity for someone who has completed relevant training and is ready to take on real-world client projects in a supportive environment.

Role Overview

As a Junior Web Developer at Enfield Dev Ltd., you will work closely with our senior developers and project managers to design, build and maintain websites and web applications for a diverse range of clients.

Key Responsibilities

  • Build responsive, accessible websites using HTML, CSS and JavaScript
  • Collaborate with designers to translate wireframes into working code
  • Write clean, well-documented and maintainable code
  • Participate in code reviews and provide constructive feedback
  • Test websites across browsers and devices for compatibility
  • Debug issues and optimise site performance
  • Attend team stand-ups and contribute to project planning
  • Continuously learn new tools and technologies

Required Skills

  • Solid understanding of HTML5 and CSS3
  • Working knowledge of JavaScript (ES6+)
  • Familiarity with version control (Git / GitHub)
  • Understanding of responsive design principles
  • Good problem-solving and communication skills
  • Willingness to learn and receive feedback
  • Eye for detail and design consistency
  • Ability to manage time and work to deadlines

Required Qualifications

  • BTEC Level 3 in Information Technology or Computing (Merit or above)
  • A-Levels including Computer Science or a related subject
  • Completion of an accredited coding bootcamp (minimum 12 weeks)
  • An apprenticeship or T-Level in Digital Technology
  • A relevant online certification (Codecademy, freeCodeCamp, etc.) supported by a portfolio
  • Equivalent international qualification in a computing-related field

Desirable (not essential)

  • Experience with a JavaScript framework (React, Vue or similar)
  • Basic knowledge of back-end technologies (Node.js, PHP or Python)
  • Familiarity with CMS platforms (WordPress, Webflow)
  • Experience with design tools (Figma, Adobe XD)
  • Understanding of basic SEO principles
